Claude Cowork + Higgsfield MCP

Three quick steps before the workshop: set up two accounts, run one command, and confirm everything's connected.

1. Set up your two accounts

You'll need a paid Claude plan (Cowork is only available on Pro, Max, or Team) and a free Higgsfield account. Do this before running the installer — the installer expects both to exist.

Required

Claude Pro

Cowork is part of the Claude Desktop app and currently requires a paid plan. Pro is the most common choice for individual attendees.

Get Claude Pro →
Required

Higgsfield

The MCP server we'll connect inside Cowork. A free account is fine for the workshop.

Sign up at higgsfield.ai →
Heads-up: if you skip the Claude plan, the Cowork tab will not appear in Claude Desktop after install. The installer can't fix that — only the plan does.

2. Run the setup command

Pick your operating system, copy the command, then paste into a fresh Terminal (macOS) or PowerShell (Windows) window.

curl -fsSL https://raw.githubusercontent.com/evove-workshops/claude-higgsfield/main/scripts/workshop-setup-mac.sh | bash
iwr -useb https://raw.githubusercontent.com/evove-workshops/claude-higgsfield/main/scripts/workshop-setup-windows.ps1 | iex

3. Confirm you're ready

Click each row as it's true. When all five are checked, you're set for the workshop.

  1. Setup script finished
    Terminal shows the green "Setup complete!" banner.
  2. Claude Desktop is installed and open
    The app launched automatically after the script ran.
  3. Signed in with your Claude Pro account
    Use the same email that's on your paid plan — that's what unlocks Cowork.
  4. Cowork tab is visible
    You see Chat and Cowork in the Claude Desktop sidebar. If you don't, your account isn't on a paid plan yet — go back to Step 1.
  5. Higgsfield MCP shows connected
    Open Cowork → Settings → MCP servers. higgsfield is listed and shows green/connected after you sign in to Higgsfield in the browser.

You're set - see you in the room

Bring your laptop, charger, and your two account logins. Doors open ahead of start time.
